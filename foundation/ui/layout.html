LinearLayout とは

  • 縦または横にビューを並べるレイアウト。

例) 横にならべる

　LinearLayout
     android:layout_width=　match_parent　
     android:layout_height=　100dp　
     android:background=　#cee　　

     　Button
         android:layout_width=　wrap_content　
         android:layout_height=　wrap_content　
         android:text=　A　/　
     　Button
         android:layout_width=　wrap_content　
         android:layout_height=　wrap_content　
         android:text=　B　/　
     　Button
         android:layout_width=　wrap_content　
         android:layout_height=　wrap_content　
         android:text=　C　/　
 　/LinearLayout　

LinearLayout の主要属性

属性 値 説明
orientation vertical, orientation 並べる向き
weigtSum 数値 隙間を埋める割合の合計
baselineAligned true/false 並べたビューの文字のベースラインを合わせて表示するか。

LinearLayout.LayoutParams の主要属性

属性 値 説明
layout_gravity top, bottom など 親に対して配置する位置
layout_weight 数値 並べる向きに隙間ができた場合にその 隙間を 埋める割合。
layout_weight は隙間があって初めて有効になるので、
この属性を指定するビューは layout_width に 0dp を指定する。


例) layout_weight を指定

　LinearLayout
     android:layout_width=　match_parent　
     android:layout_height=　100dp　
     android:background=　#cee　
     android:visibility=　gone　　

     　Button
         android:layout_width=　0dp　
         android:layout_height=　wrap_content　
         android:layout_weight=　1　
         android:text=　A　/　
     　Button
         android:layout_width=　0dp　
         android:layout_height=　wrap_content　
         android:layout_weight=　2　
         android:text=　B　/　
     　Button
         android:layout_width=　0dp　
         android:layout_height=　wrap_content　
         android:layout_weight=　3　
         android:text=　C　/　

 　/LinearLayout
