メニューの分類

  • メニューは大きく次の３種類がある。

1. オプションメニュー

  • Android バージョンで見た目が大きく変わっている。
    • Android 3.0未満
      → 端末のハードウェアのメニューボタン押下で画面下部にタイル形式で表示するメニュー。
    • Android 3.0以降
      → アクションバーにある「アクションアイテム」「オーバーフローメニュー」として表示するメニュー

2. コンテキストメニュー

  • コンテキストメニューは２種類ある。
    1. フローティングコンテキストメニュー
      → ビューを長押しするとフローティング形式で表示されるメニュー。
    2. コンテキストアクションモード
      → アクションバーの部分にメニューアイテムを持ったバー(コンテキストアクションバー)として表示するメニュー。

3. ポップアップメニュー

  • ビューの位置にポップアップ形式で表示させるメニュー。
