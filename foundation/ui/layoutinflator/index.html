レイアウトインフレータとは

  • インフレータとは XML ファイルを読み込み、その内容の Java インスタンスを生成してくれるもの。
  • インフレーターには対象となる XML によっていくつか種類がある。下記に主要なものをあげる。
    1. LayoutInflater
      レイアウトXML → View インスタンス
    2. MenuInflater
      メニューXML → MenuItem のインスタンス
    3. AnimatorInflater
      Property アニメーション XML → Animator インスタンス

レイアウトインフレータの取得

  • レイアウトインフレータのインスタンスは次の３つの方法で取得することができる。 好きなものを使えばよい。
    1. LayoutInflater の static メソッドで
      LayoutInflater inflater = LayoutInflater.from(context);
    2. システムサービスから
      LayoutInflater inflater = (LayoutInflater)context.getSystemService(Context.LAYOUT_INFLATER_SERVICE);
    3. Activity のメソッドで
      LayoutInflater inflater = activity.getLayoutInflater();
